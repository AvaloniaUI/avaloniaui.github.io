Class TopLevel

Base class for top-level windows.

Namespace:Perspex.Controls

Syntax
public abstract class TopLevel : ContentControl, IPerspexObjectDebug, INotifyPropertyChanged, ISetLogicalParent, ISupportInitialize, IContentControl, IControl, ILogical, IStyleable, INamed, IContentPresenterHost, ITemplatedControl, IPerspexObject, IInputRoot, IInputElement, IInteractive, ILayoutRoot, ILayoutable, IVisual, IRenderRoot, ICloseable, IStyleRoot, IStyleHost
Remarks

This class acts as a base for top level windows such as Window and

PopupRoot. It handles scheduling layout, styling and rendering as well as tracking the window ClientSize and IsActive state.
