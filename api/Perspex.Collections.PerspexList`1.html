Class PerspexList<T>

A notifying list.

public class PerspexList<T> : IPerspexList<T>, IList<T>, ICollection<T>, IPerspexReadOnlyList<T>, IReadOnlyList<T>, IReadOnlyCollection<T>, IEnumerable<T>, INotifyCollectionChanged, INotifyPropertyChanged, IList, ICollection, IEnumerable
Remarks

PerspexList is similar to System.Collections.ObjectModel.ObservableCollection`1 with a few added features:

It can be configured to notify the CollectionChanged event with a System.Collections.Specialized.NotifyCollectionChangedAction.Remove action instead of a System.Collections.Specialized.NotifyCollectionChangedAction.Reset when the list is cleared by setting ResetBehavior to Remove. removed A Validate function can be used to validate each item before insertion. removed
