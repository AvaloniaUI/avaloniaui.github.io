CreateCombined(IObservable<Boolean>, ReactiveCommand[])

This creates a ReactiveCommand that calls several child ReactiveCommands when invoked. Its CanExecute will match the combined result of the child CanExecutes (i.e. if any child commands cannot execute, neither can the parent)
