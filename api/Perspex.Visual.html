Class Visual

Base class for controls that provides rendering and related visual properties.

Inheritance
System.Object
PerspexObject
Animatable
Visual
Namespace:Perspex
Assembly:Perspex.SceneGraph.dll
Syntax
public class Visual : Animatable, IPerspexObject, INotifyPropertyChanged, IVisual, INamed
Remarks

The Visual class acts as a node in the Perspex scene graph and holds all the information needed for an IRenderTarget to render the control. To traverse the scene graph (aka Visual Tree), use the extension methods defined in VisualExtensions.
