Class Canvas

A panel that displays child controls at arbitrary locations.

Syntax
public class Canvas : Panel, IPerspexObjectDebug, INotifyPropertyChanged, ISetLogicalParent, ISupportInitialize, IPanel, IControl, ILogical, ILayoutable, IInputElement, IInteractive, IVisual, IStyleable, IPerspexObject, INamed, IStyleHost, INavigableContainer
Remarks

Unlike other Panel implementations, the Canvas doesn't lay out its children in any particular layout. Instead, the positioning of each child control is defined by the Canvas.Left, Canvas.Top, Canvas.Right and Canvas.Bottom attached properties.
