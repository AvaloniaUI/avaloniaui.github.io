Class Application

Encapsulates a Perspex application.

Inheritance
System.Object
Application
Namespace:Perspex
Assembly:Perspex.Application.dll
Syntax
public class Application : IGlobalDataTemplates, IGlobalStyles, IStyleRoot, IStyleHost, IApplicationLifecycle
Remarks

The Application class encapsulates Perspex application-specific functionality, including:

  • A global set of DataTemplates.
  • A global set of Styles.
  • A FocusManager.
  • An InputManager.
  • Loads and initializes rendering and windowing subsystems with InitializeSubsystems(Int32) and InitializeSubsystem(String).
  • Registers services needed by the rest of Perspex in the RegisterServices() method.
  • Tracks the lifetime of the application.
