Notifying

Gets a method that gets called before and after the property starts being notified on an object.

Remarks

When a property changes, change notifications are sent to all property subscribers; for example via the Changed observable and and the

PropertyChanged event. If this callback is set for a property, then it will be called before and after these notifications take place. The bool argument will be true before the property change notifications are sent and false afterwards. This callback is intended to support Control.IsDataContextChanging.
