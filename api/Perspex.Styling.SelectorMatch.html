Class SelectorMatch

Holds the result of a Selector match.

Remarks

There are two types of selectors - ones whose match can never change for a particular control (such as OfType<T>(Selector)) and ones whose result can change over time (such as Class(Selector, String). For the first category of selectors, the value of ImmediateResult will be set but for the second, ImmediateResult will be null and ObservableResult will hold an observable which tracks the match.
