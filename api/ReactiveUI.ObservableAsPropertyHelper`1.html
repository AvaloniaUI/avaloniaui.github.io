Class ObservableAsPropertyHelper<T>

ObservableAsPropertyHelper is a class to help ViewModels implement "output properties", that is, a property that is backed by an Observable. The property will be read-only, but will still fire change notifications. This class can be created directly, but is more often created via the ToProperty and ObservableToProperty extension methods.
