public class DirectProperty<TOwner, TValue> : PerspexProperty<TValue>, IEquatable<PerspexProperty>, IDirectPropertyAccessor where TOwner : IPerspexObject
Remarks

Direct perspex properties are backed by a field on the object, but exposed via the

PerspexProperty system. They hold a getter and an optional setter which allows the perspex property system to read and write the current value.
