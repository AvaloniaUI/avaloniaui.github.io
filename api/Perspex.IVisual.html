Interface IVisual

Represents a node in the visual scene graph.

Namespace:Perspex
Assembly:Perspex.SceneGraph.dll
Syntax
public interface IVisual
Remarks

The IVisual interface defines the interface required for a renderer to render a scene graph. You should not usually need to reference this interface unless you are writing a renderer; instead use the extension methods defined in

VisualExtensions to traverse the scene graph. This interface is implemented by Visual. It should not be necessary to implement it anywhere else.
