GetTimer(TimeSpan)

Gets a timer that fires every frame for the specified duration.

Remarks

The parameter passed to the subscriber is the progress along the animation, with 0 being the start and 1 being the end. The observable is guaranteed to fire 0 immediately on subscribe and 1 at the end of the duration.
