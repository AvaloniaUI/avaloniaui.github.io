A selector in a Style.

Remarks

Selectors represented in markup using a CSS-like syntax, e.g. "Button > .dark" which means "A child of a Button with the 'dark' class applied. The preceeding example would be stored in 3 Selector objects, linked by the Previous property:

.dark A selector that selects a control with the 'dark' class applied. > A selector that selects a child of the previous selector. Button A selector that selects a Button type.
